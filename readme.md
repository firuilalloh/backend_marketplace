# Backend Marketplace

step by step clone repo ke local
- buka terminal
- clone repo
```
git clone https://github.com/firuilalloh/backend_marketplace.git
```
- masuk ke folder backend-marketplace
```
cd backend-marketplace
```
- install dependencies
```
npm i
```
- duplicate .env copy dan ganti nama jadi .env
```
```
- run server
```
npm run dev
```
- open browser and go to http://localhost:3000
